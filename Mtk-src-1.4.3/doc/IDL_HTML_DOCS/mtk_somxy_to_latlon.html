-Abstract

   MTK_SOMXY_TO_LATLON converts from Space Oblique Mercator (SOM) X and Y coordinates to decimal latitude
   and longitude or a given MISR path.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      path      MISR path
      somx      SOM X in meters
      somy      SOM Y in meters
   
   the call:
   
      status = MTK_SOMXY_TO_LATLON( path, somx, somy, lat, lon )
   
   returns:
   
      status  0 on success; otherwise failure
      lat     decimal latitude
      lon     decimal longitude
      
-Examples

      ;;
      ;; Set up input parameters
      ;;
      path  = 230
      somx  = 17145920.0
      somy  = 222090.0
      
      status = MTK_SOMXY_TO_LATLON( path, somx, somy, lat, lon )
      
      ;;
      ;; Output...
      ;;
      print, 'path                  : ', path
      print, 'SOM X                 : ', somx
      print, 'SOM Y                 : ', somy
      print, '================================================='
      print, 'LATITUDE              : ', lat
      print, 'LONGITUDE             : ', lon 

   IDL outputs:
   
      path                  : 230
      SOM X                 : 1.71459e+07
      SOM Y                 : 222090.
      =================================================
      LATITUDE              : 26.737612
      LONGITUDE             : -54.149627
