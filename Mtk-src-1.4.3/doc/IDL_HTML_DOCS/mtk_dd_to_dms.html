-Abstract

   MTK_DD_TO_DMS converts decimal degrees to packed degrees/minutes/seconds.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      dd             decimal degrees

   the call:
   
      status = MTK_DD_TO_DMS( dd, packed_dms )
   
   returns:
   
      status         0 on success; otherwise failure
      packed_dms     packed degrees/minutes/seconds

-Examples

      ;;
      ;; Set up input parameters
      ;;
      dd          = 65.55
      status      = MTK_DD_TO_DMS( dd, packed_dms )
      
      ;;
      ;; Output...
      ;;
      print, 'decimal degrees       : ', dd
      print, '================================================='
      print, 'PACKED DMS            : ',packed_dms,FORMAT='(A,F15.3)'

   IDL outputs:
   
      decimal degrees       : 65.5500
    =================================================
      PACKED DMS            :    65033000.010

-Particulars
   None.
