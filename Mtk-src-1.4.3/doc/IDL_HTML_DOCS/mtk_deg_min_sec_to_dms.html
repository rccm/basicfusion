-Abstract

   MTK_DEG_MIN_SEC_TO_DMS converts degrees, minutes, and seconds to packed degrees/minutes/seconds.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      deg            degrees
      minutes        minutes
      seconds        seconds

   the call:
   
      status = MTK_DEG_MIN_SEC_TO_DMS( deg, minutes, seconds, packed_dms )
   
   returns:
   
      status         0 on success; otherwise failure
      packed_dms     packed degrees/minnutes/seconds

-Examples

      ;;
      ;; Set up input parameters
      ;;
      deg         = 65
      minutes     = 33
      seconds     = 0.001
      status      = MTK_DEG_MIN_SEC_TO_DMS( deg, minutes, seconds, packed_dms )
      
      ;;
      ;; Output...
      ;;
      print, 'degrees               : ', deg
      print, 'minutes               : ', minutes
      print, 'seconds               : ', seconds
      print, '================================================='
      print, 'PACKED DMS            : ',packed_dms,FORMAT='(A,F15.3)'

   IDL outputs:
   
      degrees               :       65
      minutes               :       33
      seconds               :    0.00100000
    =================================================
      PACKED DMS            :    65033000.001
