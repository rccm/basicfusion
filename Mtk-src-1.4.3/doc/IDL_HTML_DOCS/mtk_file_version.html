-Abstract

   MTK_FILE_VERSION returns the file version for a MISR product file.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      file      MISR file
   
   the call:
   
      status = MTK_FILE_VERSION( file, version )
   
   returns:
   
      status     0 on success; otherwise failure
      version    MISR file version

-Examples

      ;;
      ;; Set up input parameters
      ;;
      file  = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      
      status = MTK_FILE_VERSION( file, version )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, '================================================='
      print, 'FILE VERSION          : ',version

   IDL outputs:
   
      file                  : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
     =================================================
      FILE VERSION          : F03_0024

-Particulars
   None.
