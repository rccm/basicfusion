-Abstract

   MTK_FILE_LGID returns the local granule ID of a MISR product file.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      file      MISR file
   
   the call:
   
      status = MTK_FILE_LGID( file, lgid )
   
   returns:
   
      status  0 on success; otherwise failure
      lgid    logical granule id
      
-Examples

      ;;
      ;; Set up input parameters
      ;;
      file  = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      
      status = MTK_FILE_LGID( file, lgid )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, '================================================='
      print, 'LGID                  : ',lgid

   IDL outputs:
   
      file                  : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
     =================================================
      LGID                  : MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf

-Particulars
   None.
