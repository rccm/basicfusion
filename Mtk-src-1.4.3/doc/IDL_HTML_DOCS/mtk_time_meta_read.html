-Abstract

   MTK_TIME_META_READ retrieves time metadata from L1B2 Ellipsoid file for use with MTK_PIXEL_TIME().

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      L1B2_filename  L1B2 Ellipsoid filename

   the call:
   
      status = MTK_TIME_META_READ( L1B2_filename, timemeta )
   
   returns:
   
      status         0 on success; otherwise failure
      timemeta       time metadata structure

-Examples

      ;;
      ;; Set up input parameters
      ;;
      L1B2_filename_aa = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      L1B2_filename_an = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AN_F03_0024.hdf'
      status           = MTK_TIME_META_READ( L1B2_filename_aa, timemeta_aa )
      status           = MTK_TIME_META_READ( L1B2_filename_an, timemeta_an )
      
      ;;
      ;; Output...
      ;;
      print, 'L1B2_filename_aa      : ', L1B2_filename_aa
      print, 'L1B2_filename_an      : ', L1B2_filename_an
      print, '================================================='
      help, /struct, timemeta_aa
      help, /struct, timemeta_an

-Particulars
   The pixel time actually varies according to camera by approximately 7 minutes.
   To get the average or center pixel acquistion time, it is recommended to use
   the AN camera time metadata.
