-Abstract

   MTK_FILE_TYPE returns the file type for a MISR product file.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      file      MISR file
   
   the call:
   
      status = MTK_FILE_TYPE( file, ftype )
   
   returns:
   
      status     0 on success; otherwise failure
      ftype      MISR file type
      
-Examples

      ;;
      ;; Set up input parameters
      ;;
      file  = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      
      status = MTK_FILE_TYPE( file, ftype )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, '================================================='
      print, 'FILE TYPE             : ',ftype

   IDL outputs:
   
      file                  : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
     =================================================
      FILE TYPE             : GRP_ELLIPSOID_GM

-Particulars
   None.
