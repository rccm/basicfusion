-Abstract

   MTK_MAKE_FILENAME returns a MISR product file name given a base directory, product, camera, path, orbit,
   and version.  Pass an empty string for any unwanted fields, such as camera.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      directory       directory
      product         product name
      camera          camera name
      path            path number
      orbit           orbit number
      version         version
   
   the call:
   
      status = MTK_MAKE_FILENAME( directory, product, camera, path, orbit, version, out_filename )
   
   returns:
   
      status         0 on success; otherwise failure
      out_filename   output file name

-Examples

      ;;
      ;; Set up input parameters
      ;;
      directory   = '../Mtk_testdata/in/'
      product    = 'GRP_ELLIPSOID_GM'
      camera     = 'AA'
      path       = '037'
      orbit      = '029058'
      version    = 'F03_0024'
      
      status = MTK_MAKE_FILENAME( directory, product, camera, path, orbit, version, out_filename )
      
      ;;
      ;; Output...
      ;;
      print, 'directory             : ', directory
      print, 'product               : ', product
      print, 'camera                : ', camera
      print, 'path                  : ', path
      print, 'orbit                 : ', orbit
      print, 'version               : ', version
      print, '================================================='
      print, 'OUT FILE NAME         : ',out_filename

   IDL outputs:
   
      directory             : ../Mtk_testdata/in/
      product               : GRP_ELLIPSOID_GM
      camera                : AA
      path                  : 037
      orbit                 : 029058
      version               : F03_0024
     =================================================
      OUT FILE NAME         : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
