-Abstract

   MTK_FILE_TO_ORBIT returns the orbit number for a MISR product file.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      file      MISR file
   
   the call:
   
      status = MTK_FILE_TO_ORBIT( file, orbit )
   
   returns:
   
      status     0 on success; otherwise failure
      orbit      MISR orbit number
      
-Examples

      ;;
      ;; Set up input parameters
      ;;
      file  = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      
      status = MTK_FILE_TO_ORBIT( file, orbit )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, '================================================='
      print, 'ORBIT                 : ',orbit

   IDL outputs:
   
      file                  : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
     =================================================
      ORBIT                 :        29058

-Particulars
