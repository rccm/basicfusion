-Abstract

   MTK_FILE_COREMETADATA_QUERY returns the number of coremetadata parameters and a list of all coremetadata parameter names for a given file.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      filename       MISR product file

   the call:
   
      status = MTK_FILE_COREMETADATA_QUERY( filename, nparams, paramlist )
   
   returns:
   
      status         0 on success; otherwise failure
      nparams        number of coremetadata parameters
      paramlist      string array of all coremetadata parameter names

-Examples

      ;;
      ;; Set up input parameters
      ;;
      file        = '../Mtk_testdata/in/MISR_AM1_AS_LAND_P037_O029058_F06_0017.hdf'
      status      = MTK_FILE_COREMETADATA_QUERY( file, nparams, paramlist )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, 'nparams               : ', nparams
      print, '================================================='
      FOR i=0,MIN([5,nparams-1]) DO $
         PRINT, 'COREMETADATA PARAMETER NAME #'+STRTRIM(i+1,2)+ $
         ' = '+paramlist[i]

   IDL outputs:

      file                  : ../Mtk_testdata/in/MISR_AM1_AS_LAND_P037_O029058_F06_0017.hdf
      nparams               :           42
      =================================================
      COREMETADATA PARAMETER NAME #1 = LOCALGRANULEID
      COREMETADATA PARAMETER NAME #2 = SIZEMBECSDATAGRANULE
      COREMETADATA PARAMETER NAME #3 = PRODUCTIONDATETIME
      COREMETADATA PARAMETER NAME #4 = LOCALVERSIONID
      COREMETADATA PARAMETER NAME #5 = PGEVERSION
      COREMETADATA PARAMETER NAME #6 = MEASUREDPARAMETERCONTAINER
