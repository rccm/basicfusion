-Abstract

   MTK_FILE_GRID_TO_FIELDLIST returns a string array of fields associated with a given grid.

-Copyright

   Copyright (2005), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O
   
   Given:
   
      file      MISR file
      grid      name of grid
   
   the call:
   
      status = MTK_FILE_GRID_TO_FIELDLIST( file, grid, nfields, fieldlist )
   
   returns:
   
      status    0 on success; otherwise failure
      nfields   number of fields
      fieldlist string array of field names
      
-Examples

      ;;
      ;; Set up input parameters
      ;;
      file  = '../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf'
      grid  = 'RedBand'
      
      status = MTK_FILE_GRID_TO_FIELDLIST( file, grid, nfields, fieldlist )
      
      ;;
      ;; Output...
      ;;
      print, 'file                  : ', file
      print, 'grid                  : ', grid
      print, '================================================='
      FOR i = 0, nfields-1 DO $
         PRINT, 'FIELD #'+    $
         STRTRIM(i+1,2)+      $
         '              : '+fieldlist[i]

   IDL outputs:
   
      file                  : ../Mtk_testdata/in/MISR_AM1_GRP_ELLIPSOID_GM_P037_O029058_AA_F03_0024.hdf
      grid                  : RedBand
     =================================================
      FIELD #1              : Red Radiance/RDQI
      FIELD #2              : Red Radiance
      FIELD #3              : Red RDQI
      FIELD #4              : Red DN
      FIELD #5              : Red Equivalent Reflectance
      FIELD #6              : Red Brf
