-Abstract

    MTK_SETREGION_BY_PATH_SOM_ULC_LRC Sets a region by path and upper left and lower right SOM X/Y.

-Copyright

    Copyright (2013), California Institute of Technology.
    U.S. Government sponsorship acknowledged.

-I/O
 
    Given: 
        path        Path
        ulc_som_x   Upper left corner SOM X
        ulc_som_y   Upper left corner SOM Y
        lrc_som_x   Lower right corner SOM X
        lrc_som_y   Lower right corner SOM Y 
        
    the call:

        status = MTK_SETREGION_BY_PATH_SOM_ULC_LRC( path, ulc_som_x, ulc_som_y, lrc_som_x, lrc_som_y, region )

    returns: 
        status      0 on success; otherwise failure
        region      region set by path and SOM ULC/LRC

-Examples

    ;;
    ;; Set up input parameters
    ;;
    path        = 27
    ulc_som_x   = 15600000.0
    ulc_som_y   = -300.0
    lrc_som_x   = 16800000.0
    lrc_som_y   = 2000.0
    ;;
    ;; The call
    ;;
    status = MTK_SETREGION_BY_PATH_SOM_ULC_LRC( path, ulc_som_x, ulc_som_y, lrc_som_x, lrc_som_y, region )
    ;;
    ;; Output...
    ;;
    print, '================================================='
    help, region
