-Abstract

   MTK_CREATE_GEOGRID Creates a regularly spaced geographic 2-D grid

-Copyright

   Copyright (2013), California Institute of Technology.
   U.S. Government sponsorship acknowledged.

-I/O

   Given: 
   ulc_lat_dd	Upper left corner latitude
   ulc_lon_dd	Upper left corner longitude
   lrc_lat_dd	Lower right corner latitude
   lrc_lon_dd	Lower right corner longitude
   lat_cellsize_dd	Latitude cell size in decimal degrees
   lon_cellsize_dd	Longitude cell size in decimal degrees 
   
   the call:
   
      status = MTK_CREATE_GEOGRID(ulc_lat_dd, ulc_lon_dd, lrc_lat_dd, lrc_lon_dd, lat_cellsize_dd, lon_cellsize_dd, latbuf, lonbuf)

   returns: 
    status         0 on success; otherwise failure
    latbuf         2-D latitude buffer
    lonbuf         2-D longitude buffer   
   
   
-Examples


	;;
	;; Set up input parameters
	;;
	ulc_lat_dd = 40
	ulc_lon_dd = -120.0
	lrc_lat_dd = 30
	lrc_lon_dd = -110.0
	lat_cellsize_dd	= 0.25
	lon_cellsize_dd	= 0.25
	;;
	;; The call
	;;
	status = MTK_CREATE_GEOGRID( ulc_lat_dd, ulc_lon_dd, lrc_lat_dd, lrc_lon_dd, lat_cellsize_dd, lon_cellsize_dd , latbuf, lonbuf )
	;;
	;; Output...
	;;
	print, '================================================='
	help, latbuf, lonbuf
      
   IDL outputs:
      LATBUF          DOUBLE    = Array[41, 41]
      LONBUF          DOUBLE    = Array[41, 41]
